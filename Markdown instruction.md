## Списки

    Чтобы добавить не нумерованные списки, необходимо пункты выделить звездочкой (*), вот так:
    * Элемент 1
    * Эдемент 2
    * Элемент 3

    Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать, вот так:
    1. Элемент 1
    2. Элемент 2
    3. Элемент 3
